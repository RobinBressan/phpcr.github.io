public interface EventJournalInterface
extends \Countable

An EventJournal is an extended Iterator that provides the additional method skipTo().

All elements in this iterator are of type EventInterface.

PHPCR Note: This is the only iterator interface we kept, as it adds additional value (performance).

public void skipTo(integer date)

Skip all elements of the iterator earlier than date.

If an attempt is made to skip past the last element of the iterator, no exception is thrown but the subsequent next() will fail.

Parameters:
date - Value that represents an offset in milliseconds from the epoch.
