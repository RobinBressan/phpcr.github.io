public interface ChildNodeInterface
extends ConstraintInterface

Tests whether the selector node is a child of a node reachable by absolute path path.

A node-tuple satisfies the constraint only if selectorNode.getParent().isSame(session.getNode(path)) would return true, where selectorNode is the node for the specified selector.
