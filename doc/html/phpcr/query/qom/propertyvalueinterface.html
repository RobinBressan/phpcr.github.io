public interface PropertyValueInterface
extends DynamicOperandInterface

Evaluates to the value (or values, if multi-valued) of a property.

If, for a node-tuple, the selector node does not have a property named property, the operand evaluates to null.

The query is invalid if:

- selector is not the name of a selector in the query - property is not a syntactically valid JCR name
