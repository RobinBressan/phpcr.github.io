Class TraversingItemVisitor

TraversingItemVisitor
All Implemented Interfaces:
PHPCR\ItemVisitorInterface

public abstract class TraversingItemVisitor

An implementation of ItemVisitor.

TraversingItemVisitor is an abstract utility class which allows to easily traverse an Item hierarchy. TraversingItemVisitor makes use of the Visitor Pattern as described in the book 'Design Patterns' by the Gang Of Four (Gamma et al.). Tree traversal is done observing the left-to-right order of child Items if such an order is supported and exists.

visit

public void visit(ItemInterface item)

Called when the Visitor is passed to an Item.

It calls TraversingItemVisitor.entering(ItemInterface, int) followed by TraversingItemVisitor.leaving(ItemInterface, int). Implement these abstract methods to specify behavior on 'arrival at' and 'after leaving' the $item.

If this method throws, the visiting process is aborted.

Parameters:
item - the Node or Property that is accepting this visitor.
Throws:
\PHPCR\RepositoryException - if an error occurs.
